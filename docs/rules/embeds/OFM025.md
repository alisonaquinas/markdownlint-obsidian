# OFM025 - Embed size on non-image

**Severity:** warning
**Fixable:** yes (Phase 9)
**Added in:** Phase 5

## What it does

Warns when an embed carries a `|WIDTH` or `|WIDTHxHEIGHT` sizing hint
but its target is not an image. Obsidian ignores the hint in that
case, so author intent is lost silently. A common source of confusion
when a PDF or video embed fails to resize.

The Phase 9 autofix will strip the redundant hint.

## Bad example

```markdown
![[report.pdf|500]]
![[clip.mp4|800x600]]
```

## Good example

```markdown
![[report.pdf]]
![[photo.png|500]]
```

## Config

Disable entirely:

```jsonc
{
  "rules": {
    "OFM025": { "enabled": false }
  }
}
```
