# OFM021 - Invalid embed syntax

**Severity:** error
**Fixable:** no
**Added in:** Phase 5

## What it does

Flags two malformed embed patterns that never reach the node extractor:

1. `![[]]` with a whitespace-only body.
2. `![[target` with no closing `]]` on the same line.

Matches inside fenced code blocks and inline code spans are skipped.

## Bad example

```markdown
![[]]

![[oops
```

## Good example

```markdown
![[notes/index]]
```

## Config

Disable this rule in `.obsidian-linter.jsonc`:

```jsonc
{
  "rules": {
    "OFM021": { "enabled": false }
  }
}
```
