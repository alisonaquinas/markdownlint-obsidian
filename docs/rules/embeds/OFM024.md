# OFM024 - Disallowed embed extension

**Severity:** error
**Fixable:** no
**Added in:** Phase 5

## What it does

Reports embeds whose file extension is not in the configured
`embeds.allowedExtensions` list. The comparison is case-insensitive.
Extensionless targets are treated as markdown (the Obsidian default).

## Bad example

With the default allowList:

```markdown
![[script.exe]]
```

## Good example

```markdown
![[photo.png]]
```

## Config

The default allowList covers markdown, common image formats, PDF, and
the common audio/video formats:

```jsonc
{
  "embeds": {
    "allowedExtensions": [
      "md", "png", "jpg", "jpeg", "gif", "svg", "webp",
      "pdf", "mp4", "webm", "mp3", "wav", "ogg"
    ]
  }
}
```

Disable entirely:

```jsonc
{
  "rules": {
    "OFM024": { "enabled": false }
  }
}
```
