# OFM063 — Tag trailing slash

**Severity:** error
**Fixable:** yes (Phase 9)
**Added in:** Phase 3

## What it does

Catches the narrow case of a tag whose value ends with `/`, e.g. `#area/`.
The Phase 9 autofix simply trims the trailing slash.

## Bad example

```markdown
Note about #area/ here.
```

## Good example

```markdown
Note about #area here.
```

## Config

OFM063 requires no configuration.
