# OFM060 — Invalid tag format

**Severity:** error
**Fixable:** no
**Added in:** Phase 3

## What it does

Iterates every body-text tag the parser extracted and validates the value
against Obsidian's tag rules: A-Z, a-z, 0-9, `_`, `-`, `/`; not starting
or ending with `/`; no `//`; at least one non-digit. Most malformed tags
never reach the rule because the extractor filters them out, but
edge cases such as `#a//b` slip through and are caught here.

## Bad example

```markdown
Note about #a//b things.
```

## Good example

```markdown
Note about #a/b things.
```

## Config

OFM060 requires no configuration.
