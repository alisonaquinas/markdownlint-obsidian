# OFM065 — Mixed-case tag

**Severity:** warning
**Fixable:** yes (Phase 9)
**Added in:** Phase 3

## What it does

Warns when a tag's casing differs from the first occurrence of the same
lowercased value within the file. No-op when `tags.caseSensitive` is
true, since case differences are then meaningful.

## Bad example

```markdown
Note #project once and #Project later.
```

## Good example

```markdown
Note #project once and #project later.
```

## Config

```jsonc
{
  "tags": { "caseSensitive": false }
}
```
