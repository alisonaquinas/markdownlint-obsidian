# OFM066 — Frontmatter tag not in body

**Severity:** warning
**Fixable:** no
**Added in:** Phase 3
**Default:** disabled

## What it does

Warns when a tag listed in the file's frontmatter `tags:` array is never
referenced in the body text. Disabled by default because many vaults
intentionally tag notes via frontmatter alone.

## Bad example

```markdown
---
tags: [project, ghost]
---

Note about #project only.
```

## Good example

```markdown
---
tags: [project]
---

Note about #project only.
```
