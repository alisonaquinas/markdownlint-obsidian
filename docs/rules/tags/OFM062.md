# OFM062 — Empty tag

**Severity:** error
**Fixable:** no
**Added in:** Phase 3
**Default:** disabled

## What it does

Detects a lone `#` or `#/` token surrounded by whitespace, which is the
usual symptom of an author starting a tag and never finishing it. Lines
that look like ATX headings (`#`–`######` followed by space) are skipped.

> **Note:** Disabled by default. The detection over-fires on prose
> markdown such as docstrings or quoted prose, so it ships off and is
> available as an opt-in for vaults that benefit from it.

## Bad example

```markdown
Note with # space here.
```

## Good example

```markdown
Note with #real tag here.
```
