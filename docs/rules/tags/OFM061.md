# OFM061 — Tag depth exceeded

**Severity:** error
**Fixable:** no
**Added in:** Phase 3

## What it does

Fails when a body-text tag has more `/`-separated segments than
`tags.maxDepth` permits. Default ceiling is 5.

## Bad example

```markdown
With `tags.maxDepth = 2`:

#a/b/c is too deep.
```

## Good example

```markdown
With `tags.maxDepth = 2`:

#a/b is fine.
```

## Config

```jsonc
{
  "tags": { "maxDepth": 3 }
}
```
