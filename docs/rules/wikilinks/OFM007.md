# OFM007 - Wikilink block reference

**Severity:** warning
**Fixable:** no
**Added in:** Phase 4
**Superseded in:** Phase 6 (see OFM102)

## What it does

Warns when a block-reference wikilink (`[[page#^blockid]]`) targets a file
that does not exist in the vault. Phase 6 introduces a stronger successor
(OFM102) that additionally validates the blockid exists inside the target
file; until then this rule provides a lightweight sanity check.

## Bad example

Vault does not contain `missing.md`:

```markdown
See [[missing#^b7]] for the snippet.
```

## Good example

```markdown
See [[existing#^b7]] for the snippet.
```

## Config

OFM007 takes no options.
