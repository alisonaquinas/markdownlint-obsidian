# OFM006 - Empty wikilink heading

**Severity:** error
**Fixable:** no
**Added in:** Phase 4

## What it does

Errors when a wikilink declares a heading fragment (`#`) without any heading
text. Obsidian renders the link but it never scrolls to a meaningful
anchor.

## Bad example

```markdown
Jump to [[page#]] for the good stuff.
```

## Good example

```markdown
Jump to [[page#Overview]] for the good stuff.
```

## Config

OFM006 takes no options.
