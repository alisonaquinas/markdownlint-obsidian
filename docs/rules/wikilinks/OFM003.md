# OFM003 - Self link

**Severity:** warning
**Fixable:** no
**Default:** disabled
**Added in:** Phase 4

## What it does

Warns on wikilinks whose resolved target is the same file the link lives in.
Self-links are commonly intentional (sidebars, navigation blocks, table-of-
contents anchors) so this rule is disabled by default. Vaults that forbid
the pattern can opt in.

## Bad example

`notes/index.md`:

```markdown
# Index

Jump to [[index#Contents]].
```

## Good example

`notes/index.md`:

```markdown
# Index

Jump to [[#Contents]].
```

## Config

Enable in `.obsidian-linter.jsonc`:

```jsonc
{
  "rules": {
    "OFM003": { "enabled": true }
  }
}
```
