# OFM002 - Invalid wikilink format

**Severity:** error
**Fixable:** no
**Added in:** Phase 4

## What it does

Scans raw source lines for malformed wikilink patterns that the extractor
silently discards. Three cases are flagged:

1. Empty wikilink: `[[]]` (with optional whitespace).
2. Unclosed wikilink: `[[unclosed` with no matching `]]`.
3. Nested wikilink: `[[outer [[inner]]]]`.

Matches inside fenced code blocks and inline code spans are ignored so
documentation that describes malformed syntax does not trip the rule.

## Bad example

```markdown
See [[]] for nothing.
This [[unclosed link keeps going...
```

## Good example

```markdown
See [[existing]] for details.
```

## Config

OFM002 takes no options.
