# OFM087 — Non-string entry in tags array

**Severity:** error
**Fixable:** no
**Added in:** Phase 3

## What it does

If the file's frontmatter has a `tags:` array, every entry must be a
string. Numbers, booleans, nested maps, and arrays each fire a violation.
A non-array `tags` value is OFM083's responsibility.

## Bad example

```markdown
---
tags: [project, 42]
---

Body.
```

## Good example

```markdown
---
tags: [project, "42"]
---

Body.
```

## Config

OFM087 requires no configuration.
