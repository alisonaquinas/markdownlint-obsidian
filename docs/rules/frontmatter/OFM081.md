# OFM081 — Invalid date format

**Severity:** error
**Fixable:** no
**Added in:** Phase 3

## What it does

For each key listed in `frontmatter.dateFields`, verifies that the value
parses as ISO-8601 (`YYYY-MM-DD` or full `YYYY-MM-DDTHH:MM:SS` with
optional fractional seconds and timezone). gray-matter coerces unquoted
YAML dates to JavaScript `Date` instances; both forms are accepted.

## Bad example

```markdown
---
created: not-a-date
---

Body.
```

## Good example

```markdown
---
created: 2026-04-11
---

Body.
```

## Config

```jsonc
{
  "frontmatter": { "dateFields": ["created", "updated"] }
}
```
