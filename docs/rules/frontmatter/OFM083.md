# OFM083 — Invalid frontmatter value type

**Severity:** error
**Fixable:** no
**Added in:** Phase 3

## What it does

Checks every key in `frontmatter.typeMap` against its expected JSON type:
`string`, `number`, `boolean`, `array`, or `date`. Missing keys are
ignored — that is OFM080's job. The `"date"` type defers to the same
ISO-8601 check OFM081 uses.

## Bad example

```markdown
---
count: "three"
---

Body.
```

With `typeMap = { count: "number" }`, this fires.

## Good example

```markdown
---
count: 3
---

Body.
```

## Config

```jsonc
{
  "frontmatter": {
    "typeMap": {
      "tags": "array",
      "count": "number",
      "created": "date"
    }
  }
}
```
