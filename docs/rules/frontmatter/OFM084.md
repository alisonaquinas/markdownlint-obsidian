# OFM084 — Empty required frontmatter key

**Severity:** error
**Fixable:** no
**Added in:** Phase 3

## What it does

For each entry in `frontmatter.required`, fails if the key is present but
empty. Empty means `null`, the empty string `""`, or an empty array
`[]`. Missing keys are OFM080's responsibility.

## Bad example

```markdown
---
tags: []
author: ""
---

Body.
```

## Good example

```markdown
---
tags: [project]
author: Alison
---

Body.
```

## Config

```jsonc
{
  "frontmatter": { "required": ["tags", "author"] }
}
```
