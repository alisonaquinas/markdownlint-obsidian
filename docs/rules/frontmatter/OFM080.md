# OFM080 — Missing required frontmatter key

**Severity:** error
**Fixable:** no
**Added in:** Phase 3

## What it does

Fires once for every entry in `frontmatter.required` that is absent from a
file's parsed frontmatter. Dotted paths such as `author.name` are
supported via a left-to-right object lookup.

## Bad example

```markdown
---
author: Alison
---

Body.
```

With `frontmatter.required = ["tags"]`, the file is missing `tags`.

## Good example

```markdown
---
tags: [project]
author: Alison
---

Body.
```

## Config

```jsonc
{
  "frontmatter": { "required": ["tags", "author"] }
}
```
