# OFM085 — Duplicate frontmatter key

**Severity:** error
**Fixable:** no
**Added in:** Phase 3

## What it does

Scans `parsed.frontmatterRaw` line-by-line and reports any top-level key
that appears more than once.

> **Note:** gray-matter (via js-yaml) is strict about duplicate top-level
> keys and currently surfaces them as an OFM902 parser error before
> OFM085 has a chance to fire. The rule remains in the registry as a
> forward-compat net for any future tolerant parser.

## Bad example

```markdown
---
title: First
title: Second
---

Body.
```

## Good example

```markdown
---
title: Note
---

Body.
```

## Config

OFM085 requires no configuration.
