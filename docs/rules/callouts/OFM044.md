# OFM044 - Callout fold disabled

**Severity:** warning
**Fixable:** yes (Phase 9)
**Added in:** Phase 5

## What it does

Warns when an informational callout (`NOTE`, `INFO`, `TIP`, `HINT`) uses
a fold marker (`+` or `-`) while `callouts.allowFold` is `false`. The
rule is a no-op when fold is allowed (the default), so style-agnostic
projects never see it.

## Bad example

With `callouts.allowFold: false`:

```markdown
> [!NOTE]+ Title
> body
```

## Good example

```markdown
> [!NOTE] Title
> body
```

## Config

```jsonc
{
  "callouts": {
    "allowFold": false
  }
}
```

Disable entirely:

```jsonc
{
  "rules": {
    "OFM044": { "enabled": false }
  }
}
```
