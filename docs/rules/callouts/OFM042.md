# OFM042 - Empty callout

**Severity:** warning
**Fixable:** no
**Added in:** Phase 5

## What it does

Warns when a callout has no title and no body content. Such callouts
render as an empty colored box. When `callouts.requireTitle` is enabled,
the rule additionally warns on callouts with a body but no title.

## Bad example

```markdown
> [!NOTE]
```

With `callouts.requireTitle: true`:

```markdown
> [!NOTE]
> some content
```

## Good example

```markdown
> [!NOTE] Title
> body
```

## Config

```jsonc
{
  "callouts": {
    "requireTitle": true
  }
}
```

Disable entirely:

```jsonc
{
  "rules": {
    "OFM042": { "enabled": false }
  }
}
```
